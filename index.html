Learn Javascript Craftsmanship

  • First you learn object orientation in javascript. Most people write javascript like it's C and yet it's far from it. In fact Javascript is a special type of object oriented language that takes a special care to understand. In this course we will explain all of that to you
  • Second you learn how to utilize javascript's anonymous functions. You'll learn how to curry functions, when to use closures, and why this is totally awesome. Javascript unlike other object oriented languages (Java I'm looking at you), lets you do some really neat things. In this section we'll also talk about memoization which will speed up your functions.
  • Third you'll learn how to run a javascript project from start to finish. Most people think that all javascript belongs in one big file and that they don't need to focus on organizing functionality. Utilizing what we learn in the object oriented section as well as the anonymous function section we'll learn how to organize a simple app using javascript.
  • As a bonus You'll learn how to develop using test driven development.
